## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
yarn start

# watch mode
yarn start:dev

# production mode
$ yarn start:prod
```

## Test

```bash
# unit tests
$ yarn test

# e2e tests
$ yarn test:e2e
```

## Lints

```bash
# Check
$ yarn format:check

# Format code
$ yarn format:check

$ yarn lint
```

## Documentation

```bash
# Run it locally
$ yarn doc

# Build it
$ yarn doc:build
```
